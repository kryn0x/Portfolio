# Nidhi Kumari — Data Analyst Portfolio

A professional, fully responsive portfolio website built with pure HTML, CSS, and JavaScript (no frameworks).

**Live Demo:** [View on GitHub Pages](https://kryn0x.github.io/) *(update after deployment)*

---

## Features

- Dark theme with deep purple-to-blue gradient
- Glassmorphism cards with smooth hover effects
- Typing animation in hero section
- Scroll-reveal animations on all sections
- Animated skill bars with live percentage counters
- Clickable project cards linking to GitHub repos
- Internship documents modal (Offer Letter + Certificate)
- Clickable education card linking to report card PDF
- Clickable DBMS certification linking to Infosys certificate
- Fully responsive — mobile, tablet, desktop
- Sticky navbar with active section highlighting

## Local Setup

Just open `index.html` in any modern browser — no build step or server required.

---

## Deploy to GitHub Pages

1. Push this entire project to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select branch: `main`, folder: `/ (root)`
4. Click **Save**
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

> All file paths in this project are relative, so everything works both locally and after deployment.
